This is the source code for the **Venttraffic** website, featuring two main divisions:

1.  **Venttraffic Media**: A performance marketing agency.
2.  **Venttraffic Labs**: An innovation and AI research division.

## Quick Start

To run the website locally, ensure you have PHP installed and run the following command in this directory:

```bash
php -S localhost:8000
```

Then open your browser to: [http://localhost:8000](http://localhost:8000)

## Site Structure

- **`index.php`**: The main landing page for **Venttraffic Media**.
- **`labs.php`**: The dedicated page for **Venttraffic Labs**.
- **`assets/`**: Contains CSS, JavaScript, and Image resources.
- **`includes/`**: Reusable PHP components (Header, Footer).

## Deployment

This site is built with vanilla PHP and can be deployed to any standard LAMP/LEMP stack or shared hosting environment.
